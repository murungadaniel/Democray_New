    Section 1.3 Weighted Voting Systems

    Activity 1.3.1.

    🔗
    Notes on weighted voting systems:
    🔗
    • \(\displaystyle P_i\)
      🔗
      🔗
    • \(\displaystyle w_i\)
      🔗
      🔗
    • \(\displaystyle q\)
      🔗
      🔗
    • \(\displaystyle [q|w_1,w_2,\dots,w_n]\)
      🔗
      🔗
    🔗
    Solution.
    Weighted voting systems are when there are \(n\) voters labeled \(P_1\dots P_n\text{,}\) and each voter casts a vote with a certain weight, \(w_i\text{.}\) Think of this as having each voter being able to cast \(w_i\) votes all for the same decision. In order for a motion to pass, the sum of all the votes needs to be at least \(q\text{,}\) the quota. We need the quota to be more than 50% of the sum of all the votes and no more than the sum of all the votes, but it can be anything in between. We give a weighted voting system as \([q|w_1,w_2,\dots,w_n]\text{.}\)
    🔗
    🔗
    🔗

    🔗
    What is peculiar about each of the following weighted voting systems?
    🔗

    🔗

    Definition 1.3.1.

    🔗
    A dummy is
    🔗
    Answer.
    A voter whose vote does not make any difference on whether a motion passes or not
    🔗
    🔗
    🔗

    Definition 1.3.2.

    🔗
    A dictator is
    🔗
    Answer.
    A voter where their vote is the only reason a motion passes.
    🔗
    🔗
    🔗

    Definition 1.3.3.

    🔗
    A voter has veto power if
    🔗
    Answer.
    A voter whose must vote for a motion in order for it to pass.

    🔗
    🔗

    Activity 1.3.7.

    🔗
    In designing a weighted voting system \([q|6,5,4,3,2,1]\text{,}\) what is the largest quota \(q\) you could pick without giving veto power to anyone?
    🔗
    Answer.
    \(15\text{.}\) You need the rest of the voters to be able to pass a motion.
    🔗
    🔗

    Activity 1.3.8.

    🔗
    In the weighted voting system \([q|8,5,4,1]\text{,}\) if every voter has veto power, what is the quota \(q\text{?}\)
    🔗
    Answer.
    \(18\text{.}\) The last voter needs to have veto power, so the only way a motion passes is if the vote is unanimous.
